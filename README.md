# MyBookmark
Nest.js website

## Step by step guide

Install Node first from the node website.

Check node version
```
node -v
```

Install nest js
```
sudo npm i -g @nestjs/cli
```

Check nest version
```
nest -v
```

Create a nest js project with <b>yarn</b> package manager
```
npm new project-name
```
